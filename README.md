# YAROSLAV DIGITAL — Premium Web Portfolio

Премиальный сайт-портфолио веб-разработчика / digital-студии. Выполнен в строгом тёмном стиле с объёмными элементами, 3D металлическими звёздами, интерактивными бенто-карточками и анимациями.

## Стек технологий

- **Фреймворк:** Next.js (App Router)
- **Язык:** TypeScript
- **Стилизация:** Tailwind CSS v4
- **Анимации:** Framer Motion
- **Иконки:** Lucide React

## Конфигурация и данные

Для быстрой смены брендинга, цен и контента используются следующие файлы:

1. **Контакты и брендинг:** [`src/config/site.ts`](file:///Users/macbook/колддев/src/config/site.ts)
   - Измените имя бренда (`brandName`), описание, ссылки на Telegram, цены и статистические показатели.
2. **Основной контент:** [`src/data/site-content.ts`](file:///Users/macbook/колддев/src/data/site-content.ts)
   - Содержит все текстовые материалы секций (Hero, Bento-преимущества, Проекты, Этапы процесса, Списки тарифов, Отзывы и FAQ).

## Структура компонентов

- `Header`: Плавающая скруглённая шапка с размытием и адаптивным мобильным меню.
- `Hero`: Приветственный блок с крупным заголовком, 3D металлическими звёздами и тегами доверия.
- `LogoMarquee`: Бесконечная бегущая строка с технологическими логотипами.
- `BenefitsBento`: Bento-сетка с интерактивными превью (рейтинги, графики скорости, интеграция с Google, уведомления).
- `Projects`: Карточки кейсов с адаптивными мокапами (Apex HR, автосервис ТО, детейлинг студия, Telegram-бот).
- `Results`: Объёмные карточки показателей, наложенные друг на друга с наклоном.
- `WorkProcess`: Пошаговый интерактивный таймлайн выполнения работ.
- `Pricing`: Тарифная сетка с выделенным центральным тарифом.
- `Testimonials`: Слайдер/сетка с отзывами клиентов.
- `FAQ`: Секция частых вопросов на базе плавных аккордеонов.
- `FinalCTA`: Финальный блок призыва к действию с металлическими звёздами.
- `Footer`: Подвал с навигацией и гигантским водяным знаком бренда.

## Запуск проекта

### Установка зависимостей
```bash
npm install
```

### Запуск в режиме разработки
```bash
npm run dev
```
Сайт будет доступен по адресу: http://localhost:3000

### Сборка для Production
```bash
npm run build
npm run start
```
